CryptoHack: Introduction challenges write-up

By: Compl3x-9 On: Sat 23 September 2023
In: CryptoHack
Tags: #writeup #crypto #easy

CryptoHack is a platform full of cryptography challenges, which are focused on modern, applied solutions and algorithms instead of pure theory. The introduction challenges are very easy, and were created to guide users through the process of solving challenges and posting flags (Oh lovely flags, I adore thee!)

But, mister Compl3x, if they are that easy then why are you writing a solution to them? Glad you asked.

I'm planning to make write-ups for all the challenges in CryptoHack but I dread doing anything creative (reasons: perfectionism, worry of being incorrect and, on Wednesdays, superglue inhalation). Creating this simple write-up is like a first step on the right direction. In the website's FAQ, there's instructions about what challenges can be made public (basicly only the trivial ones), but if you wanna bitch about them you can e-mail me or something :) .

Introduction Challenges

Finding flags

This challenge teach you about flags. Idk why they're called like that (neither do I know why these "hacker games" are called Capture The Flag when it has nothing to do with getting country flags, if someone knows then I'd be glad to hear :D ). A flag is a bit of text that has a special format to make it stand out. You'll get them at the end of a challenge, and it serves as proof that you've solved it.

In CryptoHack, they follow the format crypto{something_something_Kartoffeln}, and in the description you can find the first one.

Tough challenge.

Great snakes

Coding is for freaks, and snakes too. They should be together. Probably someone, somewhere.

You need to program stuff in cryptography, and the fastest and most confortable way of doing it in this particular field is in Python. There's built-in support for large integers (I LOVE THE pow() FUNCTION), a huge amount of libraries, and most importantly: everyone uses it for their challenges, which means all write-ups are in Python. So, unless you want to rewrite all the 200 line, crazy and compact solutions that you'll be reading, go and use this language.

Here, your task is to simply download and run the python program from the challenge's description.

Network attacks

The internet can be used for more than watching TikToks or beatifuly, geniously made writeups like this one (ChatGPT, what is modesty?). In many challenges interaction with a server is needed, and the simplest and the most convenient way of doing it is through a TCP connexion, on which communication is done through text in both directions.

In the description, they also mention that their servers will always recieve and send JSON objects, which is a very simple way to format data in plain text so that it's easy to parse.

For this challenge, we need to send a JSON object with the key buy and the value flag to server socket.cryptohack.org on port 11112. An example of that json object is {"buy":"flag"}, and we can communicate with the server using the command netcat or nc.

$ nc socket.cryptohack.org 11112
Welcome to netcat's flag shop!
What would you like to buy?
I only speak JSON, I hope that's ok.

--> {"buy":"flag"}
{"flag": "go and do this challenge yourself you lazy"}

Btw, in the description there's a python script which almost solves the challenge (you only need to change a word). It'd be a good idea to save it as a template, because we tend to forget the name of that damn json or telnetlib function (at least I do).
